# Introduction #
Ce projet est une aide en vidéo au développement.
Le projet est composé de vidéo qui permet de suivre pas après pas le vrai developpement d'une projet.
Généralement les tutoriaux que l'on trouve sur internet se résumé à des cas précis loin de la réalité et bien que ce projet n'a pas l'ambition de crée un énorme projet il est quand même un peu plus volumineux qu'un simple Hello World.

Les vidéos sont réaliser avec le logiciel Open source CamStudio.
L'environement utilisé
  * MicroSoft Windows 7
  * MicroSoft Visual Studio 2010 Professionel Version d'évaluation
  * Tortoise SVN

Le projet est un simple jeu de billard.

Le langage utilisé est MicroSoft C# ( Choix pour la rapidité de développement )

Le but du projet :
**Montré l'utilisation d'un system de gestion de source (SVN).** De commenté son code au fur et à mesure.
**Refactorisé son code** Utilisation du debugger

Le projet passe sur la phase de spécification et de conception. Qui est pourtant une phase essentiel de la réalisation d'un "gros" projet. Pour les spécifications je peux m'excuse car on utilisera les règles du jeux de billard facilement trouvable sur internet.

Pour la conception, je vais utilisé une méthode agile pour définir des étapes au fur et à mesure.

Je m'excuse pour les hésitations, pour être proche de la réalité le projet n'est pas préparé a l'avance tout ou presque est fait en temps réel.

Je ne prêtant pas être un master de programmation, je suppose que certain vont criée en me voyant faire certaine chose, ou en tombant dans certain piège de débutant. Je rappel donc que ceci est juste un tutoriel pour apprend aux débutant ne prenais jamais les informations pour argent comptant c'est a vous de persévérai pour trouvé votre propre "style" de développement.

N'étant pas encore spécialiste de la méthode de développement guidé par les tests, je n'utilise pas cette méthodologie. Mais comme Visual Studio propose une interface spécifique pour la gestion des tests je rajouterai des vidéos d'exemple.
